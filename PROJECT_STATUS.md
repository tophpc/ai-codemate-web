## 🎨 设计特点

### 配色方案
- **主色：** 紫蓝渐变 (#667eea → #764ba2)
- **强调色：** 粉红渐变 (#f093fb → #f5576c)
- **背景：** 深色渐变 (slate-900 → purple-900)
- **文字：** 白色/灰色系

### 动画效果
- 渐变背景流动
- 卡片悬停上浮
- 按钮缩放
- 淡入效果
- 浮动装饰
- 时间线连接

### 响应式设计
- 移动端：< 640px（单列）
- 平板：640px - 1024px（双列）
- 桌面：> 1024px（完整布局）

- 免费额度充足

- 类似 Vercel 的功能
- 免费额度
- 简单易用

- 浏览器无法直接访问 localhost（环境隔离）
- 需要通过 curl 或服务器访问测试

### 解决方案
- 使用 `npm run build` 构建测试
- 部署到 Vercel 后测试
- 或使用 ngrok 暴露本地服务
