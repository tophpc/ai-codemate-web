## 🎨 Logo 设计

### 设计理念
Logo 融合了三大元素：
1. **代码符号** `<>` - 代表编程本质
2. **AI 大脑** 六边形 - 代表人工智能
3. **能量粒子** - 代表创新活力

### Logo 构成
```
    [能量粒子]
       ⚡
    < [AI 大脑] >
    
   AICodeMate
   智码伙伴 · 编程未来
```

### 设计元素
- **左尖括号 `<`** - 代码开始
- **右尖括号 `>`** - 代码结束
- **六边形** - AI 大脑/神经网络
- **能量粒子** - 创新与活力
- **蓝紫渐变** - 科技与未来

### 文件位置
- SVG 矢量：`/public/logo.svg`
- 最小尺寸：32px
- 安全边距：1 个 Logo 高度

## ✨ 动画规范

### 持续时间
- **快速：** 0.2s - 按钮悬停
- **中速：** 0.3s - 卡片动画
- **慢速：** 0.5s - 页面过渡
- **超长：** 0.8s - Logo 展开

### 缓动函数
```css
/* 默认 */
ease: cubic-bezier(0.4, 0, 0.2, 1)

/* 弹性 */
spring: cubic-bezier(0.68, -0.55, 0.265, 1.55)

/* 缓入 */
ease-in: cubic-bezier(0.4, 0, 1, 1)

/* 缓出 */
ease-out: cubic-bezier(0, 0, 0.2, 1)
```

### 常见动画
1. **淡入：** opacity 0→1, y 20px→0
2. **缩放：** scale 0→1
3. **悬浮：** translateY(-8px)
4. **光效：** box-shadow 扩散
